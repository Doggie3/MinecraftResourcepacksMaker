# Minecraft Resource Packs Maker

## 我的世界材质包制作工具



简易工具，用于创建我的世界材质包。



如何使用：

1. 在此处下载 [Release](https://github.com/Doggie3/MinecraftResourcepacksMaker/releases)
2. 打开MinecraftResourcepacksMaker.exe
3. 选择New
4. 选择你所想要创建材质包的根目录
5. 选择版本
6. 选择封面（可选）
7. 选择开始
8. 在右侧列表点击“+”按钮添加贴图到材质包中
9. 在左侧列表选择需要操作的材质
10. 选择左侧按钮进行编辑
11. 完成后选择工具栏上的Export
12. 选择导出的zip所在目录
13. 你的材质包完成了，现在可以使用你的材质包文件了



如何添加列表中不可用版本（*发布版本中自带1.21.11和1.21.10*）：

1. 解压游戏核心jar文件
2. 在MinecraftResourcepacksMaker.exe所在目录下的data文件夹中新建对应版本号的文件夹
3. 把核心文件中的assets文件夹复制到新建的对应版本号文件夹中
4. 现在应该可以在新建材质包的版本列表中选择该版本了

*不建议添加非正式版本的文件，不建议添加过早的文件，否则程序可能崩溃*
